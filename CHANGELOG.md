# Changelog

## v0.1.1
- ปรับหัวข้อ POS และ Orders ให้รองรับ flow จ่ายเงินครบถ้วน: Cash / PromptPay / Open Bill, บันทึกเงินรับมา-เงินทอน, คิวรอส่ง และ Sales History
- ปรับ UI Navbar, ปุ่มลอย (ตะกร้า/ส่งมอบ), และแถบเมนูให้ responsive และแยก Mobile/Desktop
- เพิ่มรายละเอียดป๊อปอัพสรุปรายการขาย (เงินรับมา/เงินทอน, สถานะจ่าย/รับสินค้า, เวลาเปิด-จ่าย-ส่งมอบ)
- ปรับ flow Open Bill → Queue → ชำระ/ส่งมอบ ให้กลับไปเข้าประวัติขายถูกต้อง
